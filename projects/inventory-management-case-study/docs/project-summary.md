# Project Summary

## Context

This project evaluated an inventory and procurement decision for Southwestern University. The goal was to minimize the cost of ordering game programs and weekly inserts while accounting for vendor price breaks, ordering costs, holding costs, and operational constraints.

## Portfolio Framing

The public version highlights the operations analytics value:

- Converting vendor pricing schedules into comparable cost options
- Applying quantity discount and EOQ-style reasoning
- Separating reusable inventory from event-specific materials
- Recalculating a practical order plan after the initial model comparison
- Translating the model output into procurement recommendations

## Decision Alternatives

The case compared two vendors:

- Quality Printing, the local vendor
- First Printing, a lower-price vendor with additional travel cost

## Interpretation

The quantity discount model showed that First Printing was less expensive for both programs and inserts. The practical recommendation adjusted the modeled result because programs and inserts have different operational constraints. Programs could be ordered in larger batches, while inserts were unique to each game and needed separate orders.

The final recommendation was to use First Printing and pursue additional cost reductions through volume negotiation, trip consolidation, annual order commitments, and just-in-time pickup.

## Tooling Note

The original classroom analysis was delivered as a Word report. The portfolio version does not emphasize the report format because the enduring value is the inventory model, vendor comparison, cost interpretation, and procurement recommendation.
